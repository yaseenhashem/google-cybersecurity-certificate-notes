# Course 01 Foundations of Cybersecurity

## Key Concepts
- **Compliance:** adhering to standards and regulations, helps organizations avoid fines and security breaches.
- **Security frameworks:** guidelines used for building plans to help mitigate risks and threats to data and privacy.
- **Security controls:** reduces specific risks, used with security frameworks to establish strong security posture.
- **Security posture:** an org.'s ability to manage defense of critical assets and data and react to change.
- **Threat actor:** malicious attacker
- **Internal threat:** employee or partner that poses security risk. Sometimes accidental.
- **Network security:** keeping network infastructure secure.
- **Cloud security:** ensuring cloud is properly configured and secure
- **Personally Identifiable Information (PII):** info that can be used to infer someone's identity (name, DOB, phone number, ip address, etc.)
- **Sensitive Personally Identifiable Information (SPII):** more damaging info than PII (SIN, medical info, etc)
